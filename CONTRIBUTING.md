# Contributing

We welcome contributions! Here's how to get started.

## Development

1. Fork the repository
2. Clone your fork
3. Run `bundle install`
4. Run `bundle exec jekyll serve --livereload`
5. Make your changes
6. Test locally
7. Submit a pull request

## Pull request guidelines

- Keep changes focused and minimal
- Update documentation (README, README-FR) if needed
- Test your changes locally before submitting
- Follow the existing code style

## Code style

- CSS: native CSS with custom properties, no preprocessors
- JavaScript: vanilla JS, no frameworks
- HTML: semantic HTML5, accessible

## Reporting issues

Open an issue on GitHub with a clear description and reproduction steps.
