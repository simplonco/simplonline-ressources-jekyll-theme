# Changelog

## 0.1.0 — Initial release

- Basic Jekyll theme structure (`_layouts`, `_includes`, `assets`)
- Single default layout with semantic HTML5
- Dark/light mode with auto-detection and manual toggle (localStorage persistence)
- Responsive design (mobile-first)
- Native CSS with custom properties (no Sass)
- Utility classes: `alert-info`, `alert-warning`, `text-center`, `text-end`
- Bilingual documentation (EN/FR)
